# angular-crop-it-now

Intro
=====

this is the client side of Crop-it NOW !

Server repository is [here](https://github.com/Berdal84/crop-it-now-server)


How to use ?
============

Clone the repo and run from the repo root directory :

"npm install"

"ng serve"
